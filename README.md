# Elite Punch Invoice

Base React/Vite/PWA mobile-first pour une application de punch, factures, employes, payes, catalogue, comptabilite et reglages.

## Stack

- React 18
- Vite
- vite-plugin-pwa
- CSS custom avec themes

## Themes inclus

- Carbon Gold
- Arctic Ledger
- Storm Steel

## Commandes

```bash
npm install
npm run dev
npm run build
```
